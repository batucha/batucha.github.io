飞机杯保养流程

发表于 2025-04-20

目录

  1. 准备工具
  2. 护理过程
    1. 清洗
    2. 擦干
    3. 护理
  3. 注意事项
  4. 推荐
  1. 准备工具
    1. 硅藻棒（任意）【重要】
    2. 杯子保护粉【重要】（我也不知道该如何称呼）
    3. 通道冲洗器（如下图）
    4. 清洁液（洗手液、沐浴露等）
    5. 干毛巾（任意）
    6. 容器【重要】
    7. 吹风机
  2. 护理过程
    1. 清洗
      1. 清洗液清洗杯子表面
      2. 简单清洗通道内部
      3. 将清洗液挤入通道中，用手指抹匀
      4. 水龙头固定好通道冲洗器
      5. 将冲洗器伸入通道内
      6. 打开水龙头进行冲洗【注意！不能堵住通道口！水在通道内积累会撑爆杯子！】
      7. 前后移动、旋转冲洗器，确保清洗到位（可用手指检查是否有黏腻感）
      8. 杯子清洗完成
    2. 擦干
      1. 将杯子表面用毛巾擦干
        • 【不能真的“擦”。未涂杯子保护粉的杯子十分脆弱。用力擦拭会损坏杯子】
        • 轻轻“按”，来去除杯子表面的水分
      2. 将硅藻棒伸入通道内
        • 需伸到底
        • 插好后，轻轻按压杯体。让内壁充分与硅藻棒接触，吸除水分【须避免硅藻棒断在通道内。硅藻棒断裂会产生粉末，下次使用会有颗粒感（不是）。总之进到尿道里或者划伤就不好了，注意不要断裂，断裂须检查好是否都将碎片取出（重新冲一遍或许是更好的选择）】
        • 通过旋转的方式取出硅藻棒。【切记不能硬拉。否则和上文中的毛巾一样，摩擦会损伤杯体】
        • 取出的硅藻棒阴干即可。若发霉，则须更换，否则会导致通道内沾染霉菌，影响二弟健康。
      3. 用吹风机吹去杯体表面水分（可选）
        • 吹风机：
          • 风力任意
          • 无温度。冷风
        • 可对着通道吹风。能进一步降低通道内水分。
      4. 杯子已擦干
    3. 护理
      1. 将杯子保护粉【均匀地】涂抹在杯子表面
        • 【通道内不需要涂抹！！！】
        • 可洒在手掌，然后涂抹均匀
        • 可直接洒在杯子上，然后涂抹均匀
        • （我是两者混用）
        • 【一定要涂抹均匀。否则下次使用会结块，影响使用】
      2. 将杯子放入容器中。盖好盖子，阴凉处保存即可。
        • 若容器有水分；之前的保护粉残留在表面上；脏了等，必须清洗后再保存
        • 推荐保留原厂包装盒。直接放在盒子内即可。
        • 可放干燥剂
  3. 注意事项
    • 保护粉真的很重要。不涂抹会导致表面出油，下次使用的手感很糟糕。杯子能用多长时间，80%都取决的是这一步（不知道爽身粉能不能替代）
    • 一定一定要确保擦干。
    • 如果硅藻棒断了，又没有吹风机。我的解决方案是那个毛巾硬怼进通道内擦拭，然后用嘴吹干。静置一会等其水分自然蒸发后，涂粉。
    • 千万别用纸巾擦。会黏在杯体上。
  4. 推荐
    仅对我使用过的杯子进行测评。我截止25年8月14日共使用过以下产品：
    3. 4.春风精灵
